# lsrp_hacking

Scaffold resource for future LSRP hacking gameplay systems.

## Included Files

- `fxmanifest.lua`: Resource manifest and dependency declaration.
- `shared/config.lua`: Shared config placeholder.
- `client/client.lua`: Minimal client bootstrap.
- `server/server.lua`: Minimal server bootstrap.

## Notes

Set `Config.Debug = true` in `shared/config.lua` to enable basic startup logging while developing the resource.
